# MobileBERT를 활용한 소녀시대 다시 만난 세계 뮤직비디오 댓글 감성 분석 프로젝트
1. 개요
이 프로젝트는 K-pop을 대표하는 명곡 소녀시대의 ‘다시 만난 세계 (Into The New World)’ 뮤직비디오에 달린 영어 댓글을 수집하고, 이를 분석하여 팬들의 감정 반응을 MobileBERT 기반 감성 분석(Sentiment Analysis) 모델로 파악하는 것을 목표로 합니다.

이 곡은 한국 대중문화에서 상징적인 의미를 가지며, 글로벌 K-pop 팬들 사이에서도 지속적으로 언급되는 곡입니다. 본 프로젝트는 전 세계 팬들이 해당 곡에 대해 어떤 감정을 표현하고 있는지를 데이터 기반으로 분석하고 시각화함으로써, 문화적·정서적 반응의 흐름을 이해하고자 합니다.
